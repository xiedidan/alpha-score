### 开发指导

#### 后端开发（FastAPI）
1. 使用异步函数（async/await）
2. 依赖注入模式
3. Pydantic 模型验证
4. SQLAlchemy ORM
5. 完善的异常处理

#### 前端开发（Vue3）
1. Composition API
2. TypeScript 严格模式
3. Pinia 状态管理
4. WebSocket 实时更新
5. 响应式设计

#### 关键特性
- 实时数据更新（WebSocket）
- 阶梯设计器（可视化编辑）
- 图表展示（ECharts）
- 配置热重载

### 注意事项
- 所有 API 需要认证
- 敏感数据加密存储
