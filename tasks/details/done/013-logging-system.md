## 📝 详细描述

### 日志配置
1. **日志级别** - DEBUG、INFO、WARNING、ERROR
2. **日志文件**
   - logs/app/app.log - 应用日志
   - logs/error/error.log - 错误日志
   - logs/trade/trade.log - 交易日志
3. **文件轮转** - 每日轮转，保留30天
4. **日志格式** - 时间、级别、模块、消息
